## Algorithm Solutions

### 구조 / Structure
- **문제 번호별 폴더 / One folder per problem id**
  - `main.py`: 풀이 코드 / solution code
  - `input.txt`: 테스트 입력 / sample(or custom) input
  - `output.txt`: 실행 결과 / program output

### 새 문제 추가 / Add a new problem
1. **문제 번호 폴더 만들기 / Create a problem folder**
   - 예: `2504/`, `6198/`, `9935/`
2. **`main.py` 작성하기 / Write `main.py`**
3. **테스트하기 / Test**

```bash
python main.py < input.txt > output.txt
```
